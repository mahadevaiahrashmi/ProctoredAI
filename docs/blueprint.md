# **App Name**: ProctoredAI

## Core Features:

- Exam Initiation: Start an AI-proctored exam session, verifying user identity, camera access and microphone access, and setting up monitoring.
- Question Navigation: Navigate through the exam, supporting various question formats (multiple choice, text entry, etc.).
- Proctoring Status: Display real-time proctoring status with alerts of potential violations based on the tool.
- Exam Submission: Securely submit the completed exam for grading.
- Automated Violation Detection: AI tool that uses the Gemini API for detection of suspicious activities during the exam such as unauthorized applications, looking away, eye movements, or other people in the room.

## Style Guidelines:

- Primary color: Deep blue (#30475E) for a professional and secure feel.
- Background color: Light gray (#E8E8E8) to provide a clean and modern backdrop.
- Accent color: Electric Blue (#749BC2) for interactive elements and highlighting important information.
- Body and headline font: 'Inter' sans-serif for a clean, neutral, and modern look, suitable for both headlines and body text.
- Use minimalist, professional icons to represent exam controls, security status, and navigation elements.
- Employ a clean and structured layout to enhance readability and focus on the test content and proctoring information.
- Subtle transitions and animations to provide feedback on interactions without being distracting during the exam.
